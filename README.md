# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/happyv288/happiness-portfolio.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
